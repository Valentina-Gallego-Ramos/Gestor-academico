Bienvenido al Gestor Académico

Acerca del Proyecto

El Gestor Académico es una aplicación web diseñada para administrar de manera eficiente la información relacionada con estudiantes, asignaturas y notas en un entorno educativo.

Entidades Administradas:

  • Estudiantes: Registro y gestión de información personal y académica de los estudiantes.
  • Asignaturas: Administración de las materias ofrecidas, incluyendo códigos, créditos y docentes.
  • Notas: Registro y seguimiento de las calificaciones obtenidas por los estudiantes.

Desarrollo en Fase 1:

En esta primera fase se ha desarrollado la base estructural y visual de una aplicacion web denominada Gestor Académico, cuyo propósito será administrar información relacionada con Estudiantes, Asignaturas y Notas.

Próxima Fase:

En la Fase 2 se implementará la lógica de manipulación de datos mediante JavaScript, permitiendo agregar, editar y eliminar registros de estudiantes, asignaturas.
